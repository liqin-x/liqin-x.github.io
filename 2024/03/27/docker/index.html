Docker 命令

1. 镜像管理

1.1 拉取镜像

  • OPTIONS：拉取镜像时的选项。
  • NAME：要拉取的镜像名称。
  • TAG：镜像的标签。
  • DIGEST：镜像的摘要。

1.2 列出镜像

  • OPTIONS：列出镜像时的选项，例如 -a 列出所有镜像。

1.3 删除镜像

  • OPTIONS：删除镜像时的选项。
  • IMAGE：要删除的镜像ID或名称。

1.4 构建镜像

  • OPTIONS：构建镜像时的选项。
  • **PATH | URL | -**：Dockerfile所在路径、URL或标准输入。

1.5 推送镜像至仓库

  • OPTIONS：推送镜像时的选项。
  • **NAME[:TAG]**：要推送的镜像名称和标签。

1.6 保存镜像至文件

  • OPTIONS：保存镜像时的选项。
  • IMAGE：要保存的镜像ID或名称。

1.7 从文件加载镜像

  • OPTIONS：加载镜像时的选项。

1.8 查找镜像

  • OPTIONS：查找镜像时的选项。
  • TERM：搜索关键词。

1.9 查看镜像历史

  • OPTIONS：查看镜像历史时的选项。
  • IMAGE：要查看历史的镜像ID或名称。

1.10 查看镜像详细信息

  • OPTIONS：查看详细信息时的选项。
  • IMAGE：要查看详细信息的镜像ID或名称。

1.11 清理未使用的镜像

  • OPTIONS：清理未使用镜像时的选项。

2. 容器管理

2.1 运行容器

  • OPTIONS：运行容器时的选项，例如端口映射、挂载数据卷等。
  • IMAGE：要运行的容器镜像。
  • COMMAND：容器启动后要执行的命令。
  • ARG：命令的参数。

2.2 列出容器

  • OPTIONS：列出容器时的选项，例如 -a 列出所有容器，**-q** 仅显示容器ID。

2.3 停止容器

  • OPTIONS：停止容器时的选项。
  • CONTAINER：要停止的容器ID或名称。

2.4 删除容器

  • OPTIONS：删除容器时的选项。
  • CONTAINER：要删除的容器ID或名称。

2.5 查看容器日志

  • OPTIONS：查看日志时的选项。
  • CONTAINER：要查看日志的容器ID或名称。

2.6 进入容器

  • OPTIONS：进入容器时的选项。
  • CONTAINER：要进入的容器ID或名称。
  • COMMAND：要在容器内执行的命令。
  • ARG：命令的参数。

2.7. 查看容器详细信息

  • OPTIONS：查看容器详细信息时的选项。
  • CONTAINER：要查看详细信息的容器ID或名称。

2.8. 导出容器文件系统内容

  • OPTIONS：导出容器时的选项。
  • CONTAINER：要导出的容器ID或名称。
  • FILENAME.tar：导出的文件名。

2.9 导入容器文件系统内容

  • OPTIONS：导入容器时的选项。
  • **FILE/URL|-**：导入的文件或URL，使用 - 表示从标准输入导入。
  • **REPOSITORY[:TAG]**：导入的镜像仓库和标签。

2.10 检查容器内进程

  • CONTAINER：要检查的容器ID或名称。
  • ps OPTIONS：ps命令的选项，例如 -ef。

2.11 显示容器端口映射

  • CONTAINER：要显示端口映射的容器ID或名称。

3. 网络管理

3.1 列出网络

  • OPTIONS：列出网络时的选项。

3.2 创建网络

  • OPTIONS：创建网络时的选项。
  • NETWORK：要创建的网络名称。

3.3 删除网络

  • NETWORK：要删除的网络名称。

3.4 连接容器至网络

  • OPTIONS：连接容器至网络时的选项。
  • NETWORK：要连接的网络名称。
  • CONTAINER：要连接至网络的容器ID或名称。

3.5 从网络断开容器

  • OPTIONS：从网络断开容器时的选项。
  • NETWORK：要断开的网络名称。
  • CONTAINER：要断开的容器ID或名称。

4. 数据卷管理

4.1 创建数据卷

  • OPTIONS：创建数据卷时的选项。
  • VOLUME：要创建的数据卷名称。

4.2 列出数据卷

  • OPTIONS：列出数据卷时的选项。

4.3 删除数据卷

  • VOLUME：要删除的数据卷名称。

4.4 查看数据卷详细信息

  • OPTIONS：查看详细信息时的选项。
  • VOLUME：要查看详细信息的数据卷名称。

4.5 清理未使用的数据卷

  • OPTIONS：清理未使用数据卷时的选项。
